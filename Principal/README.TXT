Neste projeto decidimos importar e utilizar as funções java.net.URI e java.awt.Desktop para tocar as musicas do projeto com os links que passamos quando istanciamos os objetos AlbumMusical.
java.net.URI é para importar o tipo URI e definir o link, já o java.awt.Desktop seleciona e abre o seu navegador com o link.

----- Aviso de problema tecnico -----
Quando criamos o .jar não conseguimos passar as imagens que utilizamos na GUI, logo, ao rodar as imagens não sobem para a interface.
Entretanto, se rodar a classe principal App com as imagens no local ideal a interface irá aparecer como o esperado.
